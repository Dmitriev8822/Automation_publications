# Задача 03. Модуль AI/OpenRouter

## Цель

Реализовать работу с OpenRouter: поиск свежих новостей, генерацию текста Telegram-поста и генерацию изображения или image prompt для последующей публикации.

## Работать в файлах

- `app/ai.py`
- `app/config.py` при необходимости новых настроек
- `app/schemas.py` при необходимости согласования типов
- `tests/test_ai.py`
- `docs/modules/ai.md`

## Что реализовать

1. Класс `AIClient`.
2. Методы:
   - `find_fresh_news() -> list[News]`;
   - `generate_post(news: News) -> GeneratedPost`;
   - `generate_image(post: GeneratedPost) -> ImageAsset | None`.
3. Использовать настройки:
   - `OPENROUTER_API_KEY`;
   - `OPENROUTER_MODEL`;
   - `OPENROUTER_BASE_URL`;
   - `NEWS_TOPIC`;
   - `NEWS_LANGUAGE`;
   - `POST_LANGUAGE`;
   - `MAX_NEWS_ITEMS`;
   - `POST_STYLE`;
   - `POST_MAX_LENGTH`;
   - `INCLUDE_SOURCE_LINK`;
   - `INCLUDE_HASHTAGS`;
   - `ENABLE_IMAGE_GENERATION`.
4. В ответах OpenRouter приводить данные к Pydantic-схемам.
5. При ошибках OpenRouter выбрасывать понятные исключения или возвращать пустой список там, где это безопасно.

## Интерфейсы для других модулей

`service.py` ожидает от AI-клиента такой контракт:

```python
news_items = ai_client.find_fresh_news()
generated_post = ai_client.generate_post(news)
image = ai_client.generate_image(generated_post)
```

`find_fresh_news()` должен возвращать новости в порядке приоритета публикации: самая подходящая новость первой.

## Откуда брать информацию

- Настройки: `app/config.py`.
- Общие схемы: `app/schemas.py`.
- OpenRouter API: официальная документация OpenRouter.
- Бизнес-процесс: `docs/ARCHITECTURE.md`.

## Что нельзя делать

- Не публиковать в Telegram из `ai.py`.
- Не сохранять посты в БД из `ai.py`.
- Не читать `.env` напрямую.
- Не выполнять реальные API-запросы в unit-тестах.

## Тесты

Проверить с mock/fake HTTP-клиентом:

- успешный парсинг списка новостей;
- генерацию `GeneratedPost`;
- поведение при пустом ответе;
- поведение при невалидном JSON;
- отключение изображения при `ENABLE_IMAGE_GENERATION=false`.
